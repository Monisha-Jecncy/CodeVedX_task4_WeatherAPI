# 🌦 Weather Application

A simple Weather Application built using Python, Flask, HTML, CSS, and Weather API integration.

## Features

* Search weather by city
* Real-time weather updates
* Temperature, humidity, and wind speed
* User-friendly interface

## Technologies Used

* Python
* Flask
* HTML
* CSS
* SQLite
* Weather API

## Run Project

```bash
pip install -r requirements.txt
python app.py
```

Open:

```text
http://127.0.0.1:5000
```
